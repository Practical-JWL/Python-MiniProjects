# 🎳 볼링 시뮬레이터 설치 및 환경 가이드

1. 개발 환경
- 파이썬 버전: Python 3.12.12 권장

2. 필수 라이브러리 설치
- 아래 명령어를 복사하여 터미널(CMD)에 입력하세요.
  pip install flet

3. 가상환경 설정 (선택 사항)
- 프로젝트 폴더에서 아래 명령어를 순서대로 입력하세요.
  python -m venv venv
  source venv/Scripts/activate  # Windows 환경

결과: self.controls 리스트를 직접 교체함으로써 메모리 점유율을 최적화하고, 어떠한 환경에서도 끊김 없는 사용자 경험(UX)을 제공하게 되었다.

[!NOTE]
이 과정은 복잡한 UI를 다룰 때 프레임워크의 라이프사이클을 이해하고, 효율적인 상태 관리(State Management)를 구현하는 능력을 배양하는 계기가 되었다.
